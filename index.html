reBEM is a set of independent decoupled packages for working with BEM in React ecosystem

  • rebem — create React elements with BEM props transforming into the className
  • rebem-jsx — jsx support for reBEM
  • classname — helpers for composing and parsing BEM classNames
  • css — BEM syntax for CSS
  • enzyme — reBEM addons for Enzyme
  • enzyme — reBEM addons for React Test Utilities
  • layers-loader — Webpack loader for composing sets (layers) of React components. It allows you to easily create themes and share entire component libraries
